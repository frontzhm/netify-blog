iconfont的使用

发表于 2017-11-28 | 分类于 css |

iconfont

iconfont可以让图标像字体一样修改font-size color,而且不需要图片, 可以简单理解为是一种特殊字体

iconfont的添加

一般去阿里巴巴矢量图标库
搜索图标 => 加入到购物车 => 添加到相应项目
进到具体项目 => 对图标可以进行微调 => 放大 缩小 旋转 改名字

iconfont的使用

本地使用

图标下载到本地: 进入到具体项目 => 选择下载至本地 => 解压 => 其实需要的文件一般是 iconfont开头的文件, 暂时可以不用js

网上链接

图标生成: 进入到具体项目 => 选择暂无代码点此生成 => 这样就有一个链接了

vue项目里添加

index.html里添加相应的链接 想要下载至本地的话可以放在static里

项目的管理

iconfont的项目管理, 添加成员, 项目成员可以看到相应的项目

进去具体的项目 => 更多操作

注意事项

  1. 一般iconfont用在纯色图标, 一定用多色图标的话用symbol
  2. 每次增加图标, 都需要更新链接或者重新下载
  3. 删除图标必须和组员沟通
