好好了解原理

fetch 方法的第一个参数可以是 Request 对象，也可以是一个 url，第二个参数可选，包含一些配置信息。fetch 方法返回 Promise。Promise的值是Response的实例。

Request 对象用于描述请求内容。构造函数接受的参数和 fetch 函数的参数形式一样，实际上 fetch 方法会使用传入的参数构造出一个 Request 对象来。
Response 实例有很多属性，常用的

要想从 Response 的实例中拿到最终的数据需要调用下面这些方法，这些方法都返回一个 Promise 并且使用对应的数据类型来 resolve。常用到的是json。

arrayBuffer：把响应数据转化为 arrayBuffer 来 resolve

blob：把响应数据转换为 Blob 来 resolve

formData：把响应数据转化为 formData 来 resolve

json：把响应数据解析为Object后 resolve

text：把响应数据当做String来调用 resolve
