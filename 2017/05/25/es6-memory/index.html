块级作用域

let，const作用在块级作用域。
let定义块级作用域的变量。
const定义块级作用域的常量，常量名习惯性大写，如const API = "..."
注意：只有var，function会预编译。let,const,箭头函数均不会。

箭头函数

  • 简化函数写法
  • 不改变this，换言之箭头函数里面的this就是外层代码块的this
  • 常用于回调函数中

class

  • 简化类的写法，更加直观
  • 继承更简单
  • 必须new

增强对象的字面量

  • 可以在对象字面量里定义此对象的原型
  • 直接调用父类方法
  • 定义方法可以不用function关键字
  • 合并对象Object.assign({},source1,source2)

字符串的字面量

简化字符串和变量的混合体，${varibel}

解构

定义：从数组和对象中提取值，对变量进行赋值。
请注意是对变量进行赋值。

  • 数组解构：索引一一对应（表象与顺序有关，本质还是key，因为数组的key值就是索引）。

  • 对象解构：key值必须一一对应(与顺序无关)

  • 字符串解构：拆成单个字符

Map and Set

Set类似于数组，但是会自动过滤掉重复的值，Array.from方法可以将 Set 结构转为数组。Set键名和键值是同一个值。
Map类似对象，JavaScript 的对象（Object），本质上是键值对的集合（Hash 结构），但是传统上只能用字符串当作键，而Map键可以是各种类型的值（包括对象），是一种更完善的 Hash 结构实现。

Proxy

Proxy 可以理解成，在目标对象之前架设一层“拦截”，外界对该对象的访问，都必须先通过这层拦截，因此提供了一种机制，可以对外界的访问进行过滤和改写。Proxy 这个词的原意是代理，用在这里表示由它来“代理”某些操作，可以译为“代理器”。

modules

  1. 输出export
  • export var x = 1，相当于 export {x:x}
  • 没有default的export,输出的一定是对象，函数也是一样的
  • 一般习惯写在最后。
  • 有default的话，相当于，将值赋给default,也就是输出的就是一个值不是对象
  • 对于重命名，export {x as y},输出的就是y
  1. 导入inport
  • 用{}，导入对象，属性名必须一致，import {a,b} from 'myModule'，用的时候直接a
  • 用* ，相当于把导入的值赋值给一个obj对象，用的时候obj.a
  • 对于default，因为输出的不是一个对象，所以没必要{},而且随意命名，因为值是确定的且是唯一的，import anyName from 'myModule'
  • 对于重命名，import {y as z},把y变成z

promise

Promise 是异步编程的一种解决方案，比传统的解决方案——回调函数和事件——更合理和更强大。
