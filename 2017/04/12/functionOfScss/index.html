sass的function

Posted on 2017-04-12 | In css |

函数是可返回一个Sass任何数据类型单一值的代码块。

TL;DR

  • 函数创建使用@function和@return指令，参数可以有默认值
  • 使用函数可以用关键字参数，可以用可变参数
  • mixin着重css样式，function注重返回值

创建和使用函数

创建

想想js的创建函数

sass的创建函数其实和js类似，用到function和return指令。

  • 始终只有一个值返回。返回值类型不限。
  • 函数名的破折号和下划线是可以替换的，也就是fn-name等同于fn_name
  • 参数是可选的，函数内部可以使用全局变量

使用

想想js的使用函数，fn(args)，sass的使用函数也是类似的，放在需要返回值的位置

关键字参数

现在写一个函数，根据总列数和列宽，来得到相应的值

$total:8,$col是参数，8是total的默认值，传值会覆盖不传就是默认，细节注意的是有默认值的放后面，可选参数放最后面

可变参数

函数可以通过在参数名称后面使用3个点（非省略号）来接收可变参数。该函数将从传递给它的变量参数创建一个列表。函数将使用内置的nth函数找到变量参数列表中下标为$index的值

function和mixin的区别

mixins输出行将直接编译为CSS样式。
函数则返回一个可以作为CSS属性，或可传递给另一个function或mixin的值。
想要直接输出样式，使用mixin；想执行返回值的运算，使用function。
