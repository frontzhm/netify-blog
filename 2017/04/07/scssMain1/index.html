scss语法

Posted on 2017-04-07 | In css |

SASS是一种CSS的开发工具，提供了许多便利的写法，大大节省了设计者的时间，使得CSS的开发，变得简单和可维护。

TL;DR

  • 变量：$blue:#333;，属性中使用的话#{$blue}
  • 计算：30px + 30px * $ten
  • 嵌套：基本选择器嵌套，引用父元素&
  • 注释：// 、 /*/ 、 /! */
  • 继承另外的选择器：@extend .class1;
  • 定义代码块：@mixin、@include
  • 内置颜色函数：lighten(#cc3, 10%)
  • 引用外部文件：@import "path/filename.scss";
  • 条件语句：@if lightness($color) > 30% {...}，@else
  • 循环语句：@each $member in a, b, c, d，@for，@while
  • 自定义函数：@function

基本用法

变量

SASS允许使用变量，所有变量以$开头。
如果变量需要镶嵌在字符串之中，就必须需要写在#{}之中。

计算

SASS允许在代码中使用算式：

嵌套

选择器嵌套,在嵌套的代码块内，可以使用&引用父元素

注释

/*/ 编译之后会保留在css中
// 编译之后不会保留在css中
/! */ 重要注释,即使压缩依然保留在css中

代码的重用

继承@extend

SASS允许一个选择器，继承另一个选择器。

定义一个代码块@mixin、@include

使用@mixin命令，定义一个代码块。其可以指定参数和缺省值。
使用@include命令，调用这个mixin。使用的时候，根据需要加入参数。

内置颜色函数

SASS提供了一些内置的颜色函数，以便生成系列颜色。

插入文件@import

@import命令，用来插入外部文件。
如果是css,则等同于css的import命令。
