url解析

发表于 2017-10-11 | 分类于 node ， js |

TL;DR

  • url = origin + pathname + search + hash
  • js中解析url主要利用a标签，常用的是query
  • node有url模块
  • querystring模块

url的组成

  • href是链接，https://user:pass@sub.host.com:8080/p/a/t/h?query=string&query2=string2#hash
  • origin包括协议和host，https://user:pass@sub.host.com:8080
  • protocol协议名， https:
  • auth包括username和password，user:pass
  • host包括hostname和port，sub.host.com:8080
  • path包括pathname和search，/p/a/t/h?query=string&query2=string2
  • search包括问号和query，?query=string&query2=string2
  • hash，#hash
  • protocol，search，hash，pathname全部带标点符号
  • hostname，pathname均是字面意思，host和path包括其他部分
  • url = origin + pathname + search + hash
