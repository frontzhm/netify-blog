百度地图api简单的使用

开始

官网文档
官方的开始文档

核心类 Map

文档

Map(container: String | HTMLElement, opts: MapOptions)在指定的容器内创建地图实例，之后需要调用Map.centerAndZoom()方法对地图进行初始化。未进行初始化的地图将不能进行任何操作。只是获取地理位置的话，容器可以省略。

map实例有很多方法,以下是我觉得常用到的:

  • Point大部分时候可以是{lat:2,lng:3},但centerAndZoom和panTo必须是Point的实例
  • 设初始化地图。 如果center类型为Point时，zoom必须赋值，范围3-19级，若调用高清底图（针对移动端开发）时，zoom可赋值范围为3-18级。如果center类型为字符串时，比如“北京”，zoom可以忽略，地图将自动根据center适配最佳zoom级别centerAndZoom(center: Point, zoom: Number)
  • 将地图的中心点更改为给定的点。如果该点在当前的地图视图中已经可见，则会以平滑动画的方式移动到中心点位置。可以通过配置强制移动过程不使用动画效果panTo(center: Point, opts: PanOptions)
  • 禁用双指操作缩放disablePinchToZoom()
  • 设置地图允许的级别setZoom(zoom: Number) setMinZoom(zoom: Number) setMaxZoom(zoom: Number),放大一级视图zoomIn(),缩小zoomOut()
  • 返回地图可视区域，以地理坐标表示getBounds(),返回值是一个对象{Fe:2,Ge:2,Ke:2,Le:2,Ml:{lat:3,lng:4},vl:{lat:4:lng:6}}Fe表示左下角的lat,Ge表示左下角的lng,Ke右上角的lat,Le右上角的lng,vl左下角的边界点,Ml右上角的边界点
  • 返回两点之间的距离，单位是米getDistance(start: Point, end: Point)
  • 添加/删除控件 addControl(control: Control) removeControl(control: Control)
  • 添加/删除覆盖物 addOverlay(overlay: Overlay) removeOverlay(overlay: Overlay),清除所有覆盖物clearOverlays(),得到所有覆盖物getOverlays()
  • 关于事件map.addEventListener(eventType,fn),事件类型有很多,如click,touchend,mouseout,moveend,zoomend,dragend,resize,longpress

基础类

Point此类表示一个地理坐标点。

new BMap.Point(lng: Number, lat: Number)以指定的经度和纬度创建一个地理点坐标

有一个方法,判断两个坐标点是否相等equals(other: Point),经度和纬度都相等的时候返回s

Bounds此类表示地理坐标的矩形区域。

new BMap.Bounds(sw: Point, ne: Point)创建一个包含所有给定点坐标的矩形区域。其中sw表示矩形区域的西南角，参数ne表示矩形区域的东北角

方法有很多,常用的有containsPoint(point: Point)是否包含某个点

控件类

此类是所有控件的基类，您可以通过此类来实现自定义控件。所有控件均包含Control类的属性、方法和事件。通过Map.addControl()方法可将控件添加到地图上。
new BMap.Control()创建一个控件原型实例，通过该原型实例可创建自定义控件

属性有默认的停靠位置defaultAnchor和默认的位置偏移值defaultAnchor
方法有设置和得到停靠位置setAnchor(anchor: ControlAnchor) getAnchor(anchor: ControlAnchor),设置和得到位置偏移量setOffset(offset: Size) getOffset(offset: Size),显示控件show(),隐藏控件hide(),控件是否可见isVisible()
Anchor有BMAP_ANCHOR_TOP_LEFT BMAP_ANCHOR_TOP_RIGHT BMAP_ANCHOR_BOTTOM_LEFT BMAP_ANCHOR_BOTTOM_RIGHT(上左..)

NavigationControl

此类表示地图的平移缩放控件，可以对地图进行上下左右四个方向的平移和缩放操作。
new BMap.NavigationControl(type: NavigationControlOptions)

其他控件

GeolocationControl此类是负责进行地图定位的控件，使用html5浏览器定位功能，此类继承Control所有功能
OverviewMapControl此类表示缩略地图控件
ScaleControl此类表示比例尺控件
MapTypeControl此类是负责切换地图类型的控件，此类继承Control所有功能。
PanoramaControl此类是负责切换至全景地图的控件，此类继承Control所有功能。

覆盖物类

Overlay

覆盖物的抽象基类，所有覆盖物均继承基类的方法。此类不可实例化。

常用的方法是show() hide() isVisible()

MapPanes

此类表示地图上所有覆盖物的容器集合，没有构造函数，通过对象字面量形式表示。通过Map的getPanes方法可获得该对象实例。

Marker

此类表示地图上一个图像标注
Marker(point: Point, opts: MarkerOptions) 创建一个图像标注实例。point参数指定了图像标注所在的地理位置

有很多方法: 得到和设置标注所用的图标对象getIcon() setIcon(icon: Icon)…
有事件类型,单击,双击,拖拽,移除等

Icon

此类表示标注覆盖物所使用的图标。
Icon(url: String, size: Size, opts: IconOptions)以给定的图像地址和大小创建图标对象实例

InfoWindow

此类表示地图上包含信息的窗口。
InfoWindow(content: String | HTMLElement, opts: InfoWindowOptions) 创建一个信息窗实例，其中content支持HTML内容。1.2版本开始content参数支持传入DOM结点

其他覆盖物类

折线,圆圈,多边形….

地图图层类

常用的是交通流量层

服务类

公交路线规划方案,路线导航方案,步行导航方案

Geocoder

类用于获取用户的地址解析。
方法有解析地址，例如“北京市”getPoint(address: String, callback: Function, city: String)和反地址解析。如果解析成功，则回调函数的参数为GeocoderResult对象，否则回调函数的参数为nullgetLocation(point: Point, callback: Function, options: LocationOptions)

Geolocation

返回用户当前的位置。此方法利用浏览器的geolocation接口获取用户当前位置，不支持的浏览器将无法获取。
方法有getCurrentPosition(callback: function, options: PositionOptions),返回用户当前位置。当定位成功时，回调函数的参数为GeolocationResult对象，否则为null

其他服务类

公交线路,公交站等..
