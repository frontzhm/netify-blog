vue基础篇

此文是vue文档的缩略版。

安装

  • vue兼容版本 大于等于 ie9
  • 最新稳定版2.6.6（2019-05-05）
  • 调试工具：浏览器安装Vue devtools，控制台跟console同tab
  • 最新版本<script src="https://cdn.jsdelivr.net/npm/vue"></script>
  • 声明版本<script src="https://cdn.jsdelivr.net/npm/vue@2.6.6/dist/vue.js"></script>，生产加min.js
  • 编译器就是template字符串变成js的render函数的代码
  • wepack4.0可以配置mode:'production'

介绍-vue核心也是最基本的功能

  • 声明式渲染：采用简洁的模板语法来声明式地将数据渲染进 DOM 的系统。（）,数据也可以绑定在属性上，<span v-bind:title="message">
  • 条件和循环：数据除了能绑定到DOM文本或者属性，还能绑定到DOM结构，v-if、v-for，可以添加过渡效果
  • 处理用户输入：v-on、v-model
  • 组件化应用构建：将应用想象成组件树

Vue实例

  • 一个 Vue 应用由一个通过 new Vue 创建的根 Vue 实例，以及可选的嵌套的、可复用的组件树组成
  • 所有的 Vue 组件都是 Vue 实例，并且接受相同的选项对象 (一些根实例特有的选项除外)
  • 实例上的属性和方法：data、methods、$el、$watch
  • 实例的生命周期：不要使用箭头函数。常看生命周期的图

模板语法

可以直接用render函数，可以不用模板

插值

  • 插值-文本：，留意如果节点上加了v-once的属性，则值不会更新<span v-once></span>
  • 插值-原始的html：<span v-html="rawHtml">，当rawHtml时一堆html代码的时候，注意xss攻击，不要对用户提供的内容使用插值
  • 插值-属性：<div v-bind:id="dynamicId"></div>，注意如果是null undefined false则属性都不会存在
  • 插值-可使用js表达式：1 v-bind:i="1+9"，常用的三元，计算，拼接。Math和Date可以访问。

指令

指令是带有v-的特殊特性。当表达式的值改变的时候，将其产生的连带影响，响应式的作用于DOM

  • v-bind:href="url"中，参数是href，表达式是url
  • 2.6新增动态参数v-bind:[attr]="url"，attr是href的时候，就是href属性。
  • 修饰符：参数后面用.指明的特殊后缀<form v-on:submit.prevent="onSubmit">
  • 缩写：<a :href="url" @click="fn">

计算属性和侦听器

  • computed以声明的方式创建依赖关系，基于依赖关系进行缓存，只有依赖发生变化才会重新求值，如果单纯的方法的话，只要出现就会执行函数。如果没有依赖关系，直接用method比如getNow()
  • computed默认只有getter，但可以设置setter，当vm.fullName='zhang huahua'被赋值的时候，setter就会调用
  • watch侦听器：根据数据变动而变动，同步的情况尽量用computed，但是异步或者开销大的操作可以用watch
  • 官网很棒的一个watch的例子，附在文尾

class和style的绑定

class和style是属性，通过v-bind动态修改，另一方面，vue做了增强，表达式除了是字符串还可以是数组或对象

  • class对象的时候：:class="{ active: isActive,'text-danger': hasError }"，isActive为true则类名存在，当然也可以classObj:{ active: true,'text-danger': false }定义在data里，但是常用的是定义在computed里！
  • class数组的时候，:class="[errorClass, isActive ? activeClass : '', {active:isActive}, 'ora']"，单项可以是字符串，三元，变量，对象
  • class用在组件的时候，<hi class="p"></hi>，当在一个自定义组件上使用 class 属性时，这些类将被添加到该组件的根元素上面。只有class和style会合并属性值，其他的属性会被组件根元素同属性替换。
  • style对象的时候：:style="{ color: activeColor, fontSize: fontSize + 'px' }"通常会styleObject: { color: 'red', fontSize: '13px' }可定义在data里，常用的定义在computed里
  • style数组的时候：:style="[baseStyles, overridingStyles]"多个样式对象应用到同一个对象上

条件渲染

  • v-if/v-else，放在单个元素上面，多个元素的时候，放在template上面，v-else 元素必须紧跟在带 v-if 或者 v-else-if 的元素的后面，否则它将不会被识别
  • 用key管理可复用的元素，比如用户可以选择用用户名或者邮箱登陆，不用key，造成input会复用的情况，切换的时候不能清除input，这种情况可以将两个input用不同的key。<input v-if="isUsername" placeholder="Enter your username" key="username-input"><input v-else placeholder="Enter your email address" key="email-input">
  • v-show，用法基本一样，但show控制的是display，且不支持template元素，更没有v-else
  • v-if用于不频繁的切换，v-show用于频繁的切换
  • 同一个元素上面有v-for v-if会优先解析v-for，然后展开之后再解析v-if

列表渲染

  • 数组的时候：v-for="item in items"，v-for="(item,index) in items"，in可以换成of，item是数组每一项
  • 对象的时候：v-for="value in object"，v-for="(value, key, index) in object"，value是属性值
  • key给每个节点身份，vue根据这个用于重用和排序现有元素v-for="item in items" :key="item.id"，key尽量是字符串或者数字类型
  • 数组的变异方法：pop/unshift/push/shift/splice/sort/reverse
  • 数组的替换方法，就是返回新数组，filter，concat,slice等
  • 尽量用上面的方法操作数据，而不是通过index或者length操作，
  • 如果需要index设置某项的值，Vue.set(vm.items, indexOfItem, newValue)，vm.items.splice(indexOfItem, 1, newValue)
  • 如果需要修改数组长度vm.items.splice(newLength)
  • 数组如果需要排序或者过滤才能显示，则用computed
  • <span v-for="n in 10"> </span>，表达式可以取整数
  • 如果你的目的是有条件地跳过循环的执行，那么可以将 v-if 置于外层元素 或者template上
  • v-for用在组件上的时候，可以像任何普通元素一样用 v-for

对于已经创建的实例，Vue 不能动态添加根级别的响应式属性。但是，可以使用 Vue.set(object, key, value)方法向嵌套对象添加响应式属性。或者vm.$set。

事件处理

v-on监听DOM事件，事件触发执行一定的js代码

  • 简单的可以直接写在指令里，<button v-on:click="counter += 1">Add 1</button>
  • 复杂点的，写方法名，在method里写js代码，greet写的时候，可以传入event参数，<button v-on:click="greet">Greet</button>
  • 也可以在指令里传入参数，<button v-on:click="say('hi')">Say hi</button>，当然需要event的时候<button v-on:click="warn('hi', $event)">，注意用$event

事件修饰符

  • 事件修饰符，.stop .prevent .capture .self .once .passive，stopPropagation，preventDefault，capture捕获阶段触发普通的是冒泡阶段触发，self是只有点击该元素才会触发而父元素子元素都不行，once事件只触发一次且唯一能用到自定义的组件事件的修饰符，passive移动端的时候常用在scroll事件
  • 事件修饰符的特性，可以串联v-on:click.stop.default="do"，也可以不用表达式v-on:click.stop
  • 按键修饰符，通常只绑定在keyup事件上，$event.key为相应的键的时候触发事件，v-on:keyup.enter="submit" v-on:keyup.page-down="onPageDown"，一般enter tab delete esc space up down left right f1，delete包括backspace
  • 系统修饰键，ctrl alt shift meta，meta比较特殊，不同系统对应不同的按键，mac对应command，window对应田。系统修饰键比较特殊，事件触发时修饰键必须处于按下状态。用户操作的时候，一般摁住ctrl然后点击之类的
  • exact修饰符，有且只有某个系统修饰键被摁住的时候才会触发事件，`@click.ctrl.exact=”onCtrlClick”，没有任何系统修饰键的时候才会触发，是酱紫@click.exact=”onClick”`
  • 鼠标修饰按键修饰符，配合鼠标的事件使用，left right middle

表单输入绑定

v-model指令，本质上是语法糖，其除了表单元素，还可以在自定义组件上。

语法糖

  • text 和 textarea 元素使用 value 属性和 input 事件
  • checkbox 和 radio 使用 checked 属性和 change 事件
  • select 字段将 value 作为 prop 并将 change 作为事件

注意事项：

  • v-model会忽略所有表单元素的 value、checked、selected 特性的初始值而总是将 Vue 实例的数据作为数据来源。应该在data 选项中声明初始值。
  • 需要使用输入法（中日韩等），并不会在确定之前得到更新，需要处理这个过程的话请用input事件
  • v-model用在复选框的时候，注意布尔值和数组的区别，单个绑定的时候就是布尔值表示checked属性，多个绑定的时候就是数组表示选中的value属性
  • v-model在单选框的时候，表示选中的value
  • v-model在select的时候，表示选中的option的value，没有value的话表示innerHTML，如果请选择之类的，用下面的结构

修饰符

  • lazy，在默认情况下，v-model 在每次 input 事件触发后将输入框的值与数据进行同步 (除了上述输入法组合文字时)。你可以添加 lazy 修饰符，从而转变为使用 change 事件进行同步
  • number将用户的输入值转为数值类型
  • trim 自动过滤用户输入的首尾空白字符

组件

  • 组件必须有template的属性，Vue.component('button-counter', { template: '<button></button>' })，像是自定义标签，随意使用，注意这里如果传入data，data必须是函数，以保证每个组件的data独立
  • 通过参数prop接受父组件传递过来的数据，像使用data的属性一样使用prop，<blog-post title="My journey with Vue"></blog-post>，Vue.component('blog-post', { props: ['title'], template: '<h3>vue基础篇</h3>' })，数据多的时候，可以用对象传递数据
  • 组件的template必须只有一个根元素
  • 监听子组件的事件，<blog-post v-on:enlarge-text="postFontSize += 0.1" ></blog-post>，定义组件的template<button v-on:click="$emit('enlarge-text')"> Enlarge text </button>
    事件里可以增加参数，父组件通过$event访问到被抛出这个参数值，v-on:enlarge-text="postFontSize += $event"，v-on:click="$emit('enlarge-text', 0.1)"，或者v-on:enlarge-text="onEnlargetText"再在method那边定义
  • v-model用在组件的时候，相当于v-bind:value="txt" v-on:input="txt=$event"，子组件里$emit('input')
  • slot是<alert-box> Something bad happened. </alert-box>，在子组件的template里slot标签表示alert-box里面的innerHTML
  • 动态组件，<component v-bind:is="currentTabComponent"></component>
  • 解析模板的时候注意ul li tr td，<tr is="blog-post-row"></tr>
